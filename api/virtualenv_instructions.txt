1. Install
>> python3 -m pip install --user virtualenv

2. Create virtual environment in backend
>> python3 -m venv env

3. Activate virtual environment
>> source env/bin/activate

4. Confirm by checking location of Python interpreter
>> which python

5. Returns a path that should end in something like ".../env/bin/python"
